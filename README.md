# demo-pattern
# 1.描述
    策略模式（Strategy Pattern），通俗的讲，一个问题有多种解法即策略模式。
    策略模式属于行为型模式，一个类的行为或其算法可以在运行时更改。
# 2.应用实例
## 2.1.用户级别
    会员 --> VIP --> 特约店 --> 金牌店
## 2.2.升级规则
    1)会员升VIP：
        消费额满1000 或 购买礼包商品
    2)VIP升特约店
        销售额满5000 或 邀请10个VIP级别的用户
    3)特约店升金牌店
        结算额满10000 或 邀请50个特约店级别的用户
